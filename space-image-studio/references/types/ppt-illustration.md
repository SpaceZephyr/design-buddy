# PPT 配图（16:9）

## 用途定位

演讲幻灯单页 / 知识幻灯 / 朋友圈知识图。
不是装饰图，是**知识载体**——读者扫一眼应能完整 get 到这页的核心观点 + 关键论据。

## 比例

**16:9** （1920 × 1080 px）

## 信息密度（重要：不是越少越好）

每张图至少包含以下结构化信息：

| 元素 | 数量 | 说明 |
|------|------|------|
| 主标题 | 1 | 8–18 字，画面视觉重心 |
| 副标题 / 引言 | 1 | ≤ 25 字，对主标题的补充或概念定义 |
| 关键要点 | 3–5 个 | 每条 6–14 字，必须带**编号 / 图标 / 强调色块** |
| 关键数字或对比 | 1–2 处 | 用大字号 / 不同颜色突出 |
| 视觉锚点 | 1 个 | 图形/图标/小插图，承接主题隐喻 |

⚠️ 不允许「只有一行标题 + 一行小字 + 大片空白」的极简空壳。
观点要有论据，论据要有结构。

## 设计要点

1. **左右分栏 / 上下分区**：主标题在一侧，要点列表 / 视觉在另一侧
2. **要点必须有视觉化**：编号圈 / 图标 / 颜色卡片 / 引导线
3. **配色丰富但不混乱**：1 主色 + 2 辅色 + 1 强调色（共 3–4 个有功能的颜色）
4. **色块面积差异化**：主背景 + 局部高亮卡片 / 数字背板 / 装饰几何
5. **字号梯度清晰**：标题（最大）> 副标题 > 要点 > 数字注释（4 级）
6. **一定要有一个吸睛元素**：超大数字 / 引号 / 图标 / 箭头 / 对比

## 必填字段

- ① 核心观点（一句话主标题）
- ② 副标题或引言（1 句补充）
- ③ 3–5 个关键要点（带短描述）
- ④ 1–2 个关键数据或对比
- ⑤ 视觉锚点 / 隐喻

## 禁止

- ❌ 只有一行字 + 一片空白（这是 PPT 不是 minimal art）
- ❌ 单色背景到底（即使 minimal 风格也要有局部色块或 hairline）
- ❌ 不分层级，所有字号一致
- ❌ 要点没有任何视觉化处理（纯文字 list）
- ❌ 出现页码、Logo、版权信息
