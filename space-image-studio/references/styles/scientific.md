# scientific — 学术数据

浅色底 + 严谨等距网格，论文/数据可视化感。

## Background
- 纯白 #FFFFFF 或极浅灰 #FAFAFA

## Typography
- 标题：IBM Plex Sans / Helvetica Bold
- 正文：IBM Plex Sans Regular
- 数据：IBM Plex Mono / Roboto Mono

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #FFFFFF |
| 主文字 | #111827 |
| 网格 | #E5E7EB |
| 数据蓝 | #2563EB |
| 数据红 | #DC2626 |
| 数据绿 | #059669 |
| 数据橙 | #EA580C |
| 热力梯度 | #FFEDD5 → #C2410C → #7C2D12 |

## Visual Elements
- 标准坐标轴 + 刻度
- 等距网格背景（细灰线）
- 图例（位置：右上 / 底部居中）
- 数据点带误差条（如适用）
- 热力图：连续色带

## Do
- 数据精准，单位清晰
- 颜色用于区分系列
- 字号严格分级

## Don't
- 装饰艺术化
- 3D 立体效果
- 过多颜色（≤ 5）
