# dark-atmospheric — 深色氛围

深色背景 + 霓虹高光，科技酷感。

## Background
- 渐变深：#0A0A1A → #1A0F2E（径向或对角）

## Typography
- 标题：Space Grotesk / Geist，Bold 大字
- 正文：Geist Mono Regular

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景深 | #0A0A1A |
| 卡片 | #14142B |
| 主文字 | #F0F0F8 |
| 霓虹紫 | #A855F7 |
| 霓虹青 | #06D6A0 |
| 霓虹粉 | #EC4899 |
| 边框 | #2A2A4A |

## Visual Elements
- 节点描边带发光（box-shadow 紫/青）
- 渐变线条
- 网格背景（透明度 10%）
- 数据点带光晕

## Do
- 高对比度文字
- 霓虹色仅 2 种以内
- 大量负空间

## Don't
- 纯黑背景（用深紫/深蓝）
- 暖色暖光（黄/橙）
- 浅色卡片
