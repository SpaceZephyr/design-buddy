# corporate — 商务正装

深蓝 + 金灰，投资级正式感。

## Background
- 米白 #F8F7F4 或 深蓝 #0F172A（双模）

## Typography
- 标题：Playfair Display 或 Source Serif Pro，衬线
- 正文：Source Sans Pro / Inter，无衬线

## Color Palette
| 角色 | Hex |
|------|-----|
| 主背景 | #0F172A |
| 强调底 | #1E293B |
| 主文字 | #F8FAFC |
| 金色点缀 | #D4AF37 |
| 蓝灰 | #94A3B8 |
| 次强调 | #3B82F6 |

## Visual Elements
- 细金线分割
- 衬线大标题
- 数据柱状图用蓝→金渐变
- 角落小 logo 位（不放 logo，留位置感）

## Do
- 庄重对称构图
- 金色仅做点缀
- 字距宽松大气

## Don't
- 卡通图标
- 高饱和暖色
- 圆润手绘感
