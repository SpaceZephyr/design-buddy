# bold-editorial — 大字号杂志

Keynote / 时代周刊封面式，大字号 + 高对比。

## Background
- 单色饱和：明黄 #FACC15 / 朱红 #DC2626 / 深蓝 #1E40AF / 黑 #0A0A0A

## Typography
- 标题：Tiempos Headline / Bebas Neue / Anton，Black weight，超大
- 正文：Inter Bold

## Color Palette
- 二色撞色：背景饱和色 + 黑/白文字
- 点缀色 1 个，对比互补

## Visual Elements
- 标题字号 ≥ 画面高 1/4
- 字符可裁切出血
- 大块色块切割画面
- 极少装饰，文字即设计

## Do
- 字体粗细极端对比（Black vs Light）
- 留出块面让字呼吸
- 颜色撞色但和谐

## Don't
- 多元素堆叠
- 弱化标题
- 渐变色
