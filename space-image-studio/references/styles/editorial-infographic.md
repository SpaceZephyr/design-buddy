# editorial-infographic — 信息图编辑设计

The Economist / NYT 风格信息图，理性又美观。

## Background
- 米白 #F5F2EC 或 浅灰 #F1F5F9

## Typography
- 标题：Tiempos Headline 或 Source Serif Pro，衬线 Semi-bold
- 正文：IBM Plex Sans Regular
- 数据：IBM Plex Mono

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #F5F2EC |
| 主文字 | #1F2937 |
| 编辑红 | #DC2626 |
| 数据蓝 | #1D4ED8 |
| 中性灰 | #6B7280 |
| 强调橙 | #EA580C |

## Visual Elements
- 数据可视化：柱/折线/面积图，设计精致
- 标注线连接数据点和说明
- 小标题分区
- 引用块用大号引号

## Do
- 数据带来源注释
- 颜色用于功能区分非装饰
- 严谨构图（栅格对齐）

## Don't
- 卡通元素
- 3D 图表
- 过度装饰
