# sketch-notes — 手绘笔记

暖色 + 手绘线条，教育亲和力。

## Background
- 米色 #FAF6EC 或 浅黄 #FEF3C7，可加纸张纤维纹理

## Typography
- 标题：手写体，如 Caveat / Patrick Hand / Permanent Marker
- 正文：Kalam / Comic Neue

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #FAF6EC |
| 主线条 | #44403C（深咖） |
| 高亮 1 | #F59E0B（琥珀） |
| 高亮 2 | #EF4444（番茄） |
| 高亮 3 | #10B981（薄荷） |

## Visual Elements
- 线条：手绘抖动 1.5–2px
- 节点：手绘云形/圆角矩形
- 涂鸦元素：箭头、星星、对勾、叹号
- 重点用马克笔涂色块

## Do
- 标题手写大字
- 多用箭头、连线、强调圈
- 配色温暖

## Don't
- 绝对直线 / 数学曲线
- 冷色 / 灰色基调
- 几何严谨感
