# watercolor — 水彩柔和

水彩质感 + 暖色，艺术与生活感。

## Background
- 米色 #FAF7F2 或 浅水彩晕染

## Typography
- 标题：Cormorant Garamond / 思源宋体 Italic
- 正文：Source Sans / 思源黑体 Light

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #FAF7F2 |
| 主文字 | #3F3A36 |
| 玫瑰粉 | #E8B4B8 |
| 薄荷绿 | #B8D4C8 |
| 雾蓝 | #B8C8D4 |
| 麦秆黄 | #E8D9B8 |

## Visual Elements
- 节点用水彩晕染色块
- 边缘自然散开（非硬边）
- 极细手绘线
- 配以小植物/树叶/花瓣点缀
- 透明度叠加

## Do
- 颜色淡雅低饱和
- 水彩纹理质感
- 留白与色块呼吸交替

## Don't
- 数字感强的几何
- 高饱和强对比
- 工业感图标
