# notion — 干净 SaaS

白底 + Notion 灰，干净的产品向几何风。

## Background
- #FFFFFF 或 #FAFAFA

## Typography
- 标题：Inter / Söhne Bold
- 正文：Inter Regular

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #FFFFFF |
| 卡片 | #F7F6F3 |
| 主文字 | #37352F |
| 次文字 | #787774 |
| 蓝色强调 | #2383E2 |
| 灰边框 | #E9E9E7 |

## Visual Elements
- 卡片：圆角 8px，浅灰底
- 1px 边框
- 极少阴影（box-shadow 0 1px 2px @ 5%）
- 图标用 Lucide / Tabler 风格线性

## Do
- 用卡片组织信息块
- 适度色块区分类别
- 图标尺寸统一

## Don't
- 渐变背景
- 强烈阴影
- 多种字体
