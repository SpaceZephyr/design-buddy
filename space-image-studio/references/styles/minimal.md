# minimal — 极简留白

Apple Keynote 式克制美学，留白即设计。

## Background
- 纯白 #FFFFFF，无纹理无渐变

## Typography
- 标题：SF Pro Display / Inter，Light–Medium，大字号、宽字距
- 正文：同字族 Light，行距宽松

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #FFFFFF |
| 主文字 | #1A1A1A |
| 次文字 | #6B7280 |
| 单点缀 | #2563EB（蓝）或 #DC2626（红）任选其一 |
| 分隔线 | #E5E7EB |

## Visual Elements
- 单一点缀色，慎用
- 极细分隔线 1px
- 上下左右留白 ≥ 15%
- 简单几何形（圆/线/点）
- 数据可视化：单色或灰阶

## Do
- 用空白塑造焦点
- 文字 ≤ 10 字一句
- 仅一个层级对比

## Don't
- 多色点缀
- 装饰图标
- 阴影 / 渐变 / 立体
