# blueprint — 工程图纸

深蓝底 + 白色细线，技术与精准感。

## Background
- 深蓝 #0A2540，叠加 1px 白色网格 透明度 8%

## Typography
- 标题：JetBrains Mono / IBM Plex Mono Bold，全大写
- 正文：Inter Regular，等宽数字

## Color Palette
| 角色 | Hex |
|------|-----|
| 背景 | #0A2540 |
| 网格 | #FFFFFF @ 8% |
| 主线条 | #E2E8F0 |
| 强调线 | #06B6D4（青） |
| 节点框 | 白色 1.5px 描边 |
| 文字 | #FFFFFF |

## Visual Elements
- 节点：圆角矩形 + 1.5px 白描边
- 连线：直角折线，端点带小箭头
- 标注：尺寸标注线 + 端点小竖线
- 角落坐标：用 X/Y 轴或刻度装饰

## Do
- 信息层级用线宽区分
- 节点对齐到隐形网格
- 关键路径用青色高亮

## Don't
- 曲线 / 手绘感
- 多种字体混用
- 暖色（橙/红/黄）
