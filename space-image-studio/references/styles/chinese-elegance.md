# chinese-elegance — 中式美学

留白 + 宋体 + 中式色，文化与文人气。

## Background
- 米白 #F4EFE6 / 宣纸 #EFE9DC / 淡墨 #2D3142（暗版）
- 可加极淡墨韵纹理

## Typography
- 标题：思源宋体 / 方正清刻本悦宋 Bold
- 正文：思源宋体 Regular
- 可加竖排标题作为装饰

## Color Palette
| 角色 | Hex |
|------|-----|
| 宣纸底 | #F4EFE6 |
| 墨黑 | #1C1C1C |
| 朱砂 | #C8553D |
| 黛青 | #4A6670 |
| 鹅黄 | #E8C547 |
| 松烟 | #2D3142 |

## Visual Elements
- 大量留白（≥ 35%）
- 印章风格点缀（朱砂方印）
- 极简墨线
- 可有竖向分隔线
- 标题位置偏左或顶部，留底部呼吸

## Do
- 字距宽松、行距大
- 仅 1 个朱砂点缀
- 文字简练、有韵

## Don't
- 卡通插图
- 高饱和荧光色
- 西式装饰元素（雪花/蝴蝶结）
- 现代图标
